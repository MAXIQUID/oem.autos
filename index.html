FOE (Factory Original Equipment) Standard

The FOE schema provides a modular, VIN-native structure for automotive parts:

{
  "vin": "WBA8E9G5XGNU12345",
  "part_number": "230477",
  "category": "Brake System > Rotors",
  "condition": "Used - Excellent",
  "images": ["rotor_front.jpg", "rotor_back.jpg"],
  "timestamp_removed": "2026-01-31T14:22:00Z",
  "technician": "John Doe",
  "marketplace_status": "Listed"
}
